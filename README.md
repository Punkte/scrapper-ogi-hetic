# Scrapper OGI HETIC

## Connexion 

Pour pouvoir se connecter il faut renseigner les deux variables suivantes dans un fichier `.env` placé à la racine du repertoire

```
HETIC_USERNAME=adresseMail
HETIC_PWD=motDePasse
```

## Installer les dépendances

```bash
npm i
```

## Lancer le scrapper

```bash
npm start
```
